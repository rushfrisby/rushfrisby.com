PNGOUT an Entire Directory in Windows
Thursday, October 29, 2015

I couldn’t find a good tool for running pngout on an entire directory so I wrote one real quick. A couple of things to note. The command line syntax is:

pngsout.exe

The path to pngout.exe is defined in pngsout.exe.config so you must edit that before you start!

You must have .NET Framework 4.6 installed.
