DigitalBlasphemy.com Updater for Windows
Thursday, February 7, 2013

The Digital Blasphemy Updater is a Windows application that will update your wallpaper to the latest DigitalBlasphemy.com image. The application uses your Digital Blasphemy credentials in order to download images, so you must have a membership to use it.

You can use Windows Task Scheduler to run the EXE file whenever you want to check for new images, or run it manually.

  • Requires .NET Framework 4.0 (most PCs already have this)
